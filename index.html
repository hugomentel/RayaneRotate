OlgaRotate

Plugin to add rotate function using jquery

Download this project as a .zip file Download this project as a tar.gz file

How to use

//HTML

<div id="myDiv"></div>

//JAVASCRIPT

$("#myDiv").olgaRotate();

//CSS

.olgarotate-api-rotate-symbol{
    //You just need to put the rotate background-image
    background-image: url('http://www.draw4free.com/help/icons/rotate.png');
}

P.S: You also can also use this plugin using options. e.g:

//also can be right and center
$("#myDiv").olgaRotate({align:'left'});

//This can be used when the 
//object #myDiv also uses draggable jquery.
//default value is false.
$("myDiv").olgaRotate({updatableposition:true});
